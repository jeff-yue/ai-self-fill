⚡ uv环境文件
════════════════════════════════════════════════════════════

批处理文件：
  ⭐⭐⭐⭐⭐ 使用uv安装.bat
      → 一键安装uv和所有依赖
      → 会自动安装uv（如果未安装）

  ⭐⭐⭐⭐⭐ 使用uv运行.bat
      → 运行脚本
      → 自动使用虚拟环境

  ⭐⭐⭐⭐ 测试uv环境.bat
      → 测试uv是否安装和配置正确
      → 显示详细的环境信息

配置文件：
  ⭐⭐⭐⭐ pyproject.toml
      → uv项目配置文件
      → 定义依赖和项目元数据

  ⭐⭐⭐ .python-version
      → 指定Python版本（3.11）
      → uv会自动使用此版本

  ⭐⭐ .gitignore
      → Git忽略文件配置
      → 自动忽略虚拟环境、缓存等

自动生成：
  .venv/              - 虚拟环境目录（自动创建）
  uv.lock            - 依赖锁文件（自动生成）

🎯 使用流程总结
════════════════════════════════════════════════════════════

方式一：使用启动器（推荐新手）
  1. 双击 启动脚本.bat
  2. 选择 [2] 安装依赖
  3. 修改 config.json
  4. 选择 [1] 运行脚本

方式二：使用批处理（最快）
  1. 双击 使用uv安装.bat
  2. 修改 config.json
  3. 双击 使用uv运行.bat

方式三：使用命令行
  1. 运行 uv sync
  2. 修改 config.json
  3. 运行 uv run python run_with_config.py
