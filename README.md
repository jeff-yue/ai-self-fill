# 自助提报自动化脚本

⚡ **使用uv环境，快速、现代、可靠！**

## 功能说明

本脚本用于自动化调用自助提报API接口，实现以下功能：

1. **获取会话ID**：调用获取会话ID接口，获取conversationId
2. **批量处理消息**：从Excel文件的【输入】列读取内容，调用发送对话消息接口
3. **结果回写**：将API响应的content内容写回Excel文件的【实际输出】列

## Excel文件格式要求

Excel文件应包含以下列：

- **输入** - 必需列，包含要发送的消息内容：多轮会话拼接“say”，无say则为单轮会话
- **实际输出** - 可选列，如果不存在会自动创建，用于存储API响应结果

示例：

| 输入 | 实际输出 |
|------|----------|
| 全勤，未请假 | （API响应内容） |
| 请假1天 | （API响应内容） |

## 注意事项

1. **Excel文件备份**：建议在运行脚本前备份Excel文件
2. **网络连接**：确保网络连接正常，能够访问API接口
3. **认证token**：确保X-AUTH token有效且未过期
4. **数据格式**：输入列不应有空值，空值会自动跳过
5. **会话保持**：所有行数据使用同一个会话ID进行处理，支持多轮对话

## 错误处理

脚本包含完善的错误处理机制：

- API调用失败时会记录错误信息到日志和Excel
- 单行数据处理失败不会影响其他行的处理
- 网络超时设置为30秒
